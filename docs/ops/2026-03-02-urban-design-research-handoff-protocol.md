# OpenPlan — Urban Design Research Handoff Protocol (Canonical)

**Date (PT):** 2026-03-02  
**Owner:** COO lane (Bart) + Urban Design lane (Camila)

## Required Handoff Format (every update)
1. **Topic**
2. **Artifact path**
3. **Commit**
4. **Decision deltas**
5. **Risks/asks**
6. **Manual-verification items**

## Mandatory Rules
- Surface **scope-drift risk up front** in each handoff.
- Distinguish **ship-critical UX/design risk** from cosmetic polish.
- If external references/standards cannot be verified automatically, list as **manual-verification item**.
- State lock posture explicitly (e.g., no-new-feature lock).

## Governance Note
- Research/design strategy artifacts do **not** authorize implementation.
