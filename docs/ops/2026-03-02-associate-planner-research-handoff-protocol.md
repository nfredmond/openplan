## Required Handoff Format (every update)
1. **Topic**
2. **Artifact path**
3. **Commit**
4. **Decision deltas**
5. **Risks/asks**
6. **Manual-verification items**

## Mandatory Rules
- Surface **scope-drift risk up front** in each handoff.
- Tie findings to **pilot acceptance criteria** and planner workflow impacts.
- For any external-facing claim, include citation/evidence paths.
- State lock posture explicitly (e.g., no-new-feature lock).

## Governance Note
- Research/planning artifacts do **not** authorize implementation under lock.
